용어 중심으로 정리합니다. 동일 의미의 용어가 여러 개라면 de facto 표준을 채택합니다.

- 인프라/플랫폼 용어는 `영어 | 한국어(표준) | 설명 | 동의어/비표준 | AWS/GCP/Azure` 형식을 사용합니다.
- 개발 일반 용어는 `영어 | 한국어(표준) | 설명` 형식을 사용합니다.
- `한국어(표준)`은 팀 문서/대화에서 우선 사용하는 표현입니다.
- `동의어/비표준`은 팀에서 혼용되는 표현만 기록하고 없으면 `-`로 둡니다.
- `AWS/GCP/Azure`는 벤더별 공식 서비스명을 적습니다.

## 컴퓨트

| ------------------- | ------------- | -------------------------------------------------------- | ------------------------ | --------------------------------------------------------------- |

| Instance Type       | 인스턴스 타입 | vCPU, 메모리, 네트워크 성능을 묶은 사양 단위.            | 머신 타입, VM 사이즈     | Instance Type / Machine Type / VM Size                          |
| Auto Scaling        | 오토스케일링  | 부하에 따라 인스턴스 수를 자동으로 늘리거나 줄이는 기능. | 오토스케일, 자동 확장    | Auto Scaling / Autoscaler(MIG) / Autoscale                      |

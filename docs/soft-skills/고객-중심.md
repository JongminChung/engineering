# 고객 중심

고객 중심적인 사고를 가지고 일하는지 확인할 것.

- 고객의 이슈 파악 및 해결.
  - 책임감을 가지고 주기적으로 고객의 요구와 이슈를 파악하려는 시도 (유지보수).
- 고객의 요구 파악 및 기대치 만족 또는 뛰어 넘기.
- 꼭 기업 외부 고객일 필요는 없고 사내 다른 팀의 직원도 고객일 수 있음.

고객 중심 역량 파악 질문 (self question)

- 외부 또는 내부 고객의 요구를 만족 시켰던 서비스를 제공한 경험이 있나요?
- 또는 반대로 외부 또는 내부 고객의 기대에 만족하는 서비스를 제공하기 힘들었던 경험이 있나요?
  - 다시 그런 업무를 하게된다면 어떻게 다르게 접근할 수 있는지?
- 고객이 가지고 있는 문제를 해결하는 업무를 해본 경험이 있나요?

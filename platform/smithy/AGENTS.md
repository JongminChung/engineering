# AGENTS.md

## 범위

- Smithy 기반 샘플/PoC 프로젝트 규칙입니다.

## 구조

- 주요 프로젝트는 `full-stack-application` 아래에 위치합니다.
- 규칙/실행 방법은 `full-stack-application/README.md`와 `Makefile`을 우선합니다.

## 작업 원칙

- Smithy 모델 변경 시 관련 생성물 여부를 확인하고, 자동 생성물은 요청 없이는 수정하지 않습니다.
- 스크립트는 `full-stack-application/scripts`의 기존 흐름을 유지합니다.
